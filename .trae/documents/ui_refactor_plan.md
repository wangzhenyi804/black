# 前端界面现代化重构计划 (V2)

## 1. 目标
根据最新的原型图，将现有的界面重构为高质感的暗色系 (Dark Theme) 界面。重点在于提升数据可视化的精致度、布局的层次感以及交互的现代感。

## 2. 视觉设计规范
*   **配色方案**：
    *   **主背景**：`#0f0f0f` (深木炭黑)
    *   **卡片/容器背景**：`#18181b` (深灰黑)
    *   **侧边栏背景**：`#121212`
    *   **主色调**：电光蓝 (`#3b82f6`)、翠绿 (`#10b981`)、琥珀色 (`#f59e0b`)、紫色 (`#8b5cf6`)，用于区分不同维度的指标。
    *   **文字**：标题纯白 (`#ffffff`)，正文浅灰 (`#a1a1aa`)，提示文字深灰 (`#52525b`)。
*   **圆角与间距**：
    *   容器圆角：使用 `rounded-2xl` (1rem) 或 `rounded-3xl`。
    *   布局间距：使用更紧凑的 `gap-4` 和 `p-6`，减少无效空白。
*   **阴影与边框**：
    *   移除大部分显眼的边框，改用极细的深色边框 (`border-white/5`) 或深色投影。

## 3. 核心模块重构

### 3.1 侧边栏 (Sidebar) 重构
*   **Logo**：更新为 "BlackAd" 文字配合现代化的指南针或信号图标。
*   **导航项**：
    *   默认状态：灰色文字，无背景。
    *   激活/悬停状态：使用带有圆角的背景色块，文字高亮。
    *   图标：统一使用 `lucide-react`，并根据状态调整透明度。

### 3.2 顶部导航 (Top Header) 重构
*   **布局**：左侧留空，中间放置药丸型 (Pill-shaped) 搜索框。
*   **功能区**：右侧包含通知铃铛、深/浅色模式切换按钮、用户圆形头像（带首字母或图片）。

### 3.3 总览/概览页面 (`Dashboard.tsx` & `Insights`)
*   **标题区**：左上角显示页面标题（带图标）和副标题。
*   **指标卡片**：
    *   采用大数字展示。
    *   左上角放置带背景色的功能图标（如蓝色背景的箱子、绿色背景的增长箭头等）。
*   **统计图表**：
    *   调整 `recharts` 配置，使用深色轴线、虚线网格和发光折线。
    *   空数据状态：显示简洁的 "No data yet" 提示。

### 3.4 列表与表格重构
*   表格行改为带圆角的卡片样式，或极简的无边框行。
*   操作按钮（如“详情”、“修改”）采用胶囊风格。

## 4. 技术实施
1.  **全局 CSS**：修改 `index.css`，配置暗色主题变量及滚动条样式。
2.  **布局组件**：更新 `Layout.tsx` 的结构和样式。
3.  **页面更新**：按 `Dashboard` -> `Media` -> `CodeSlots` -> `Stats` 的顺序逐步适配。
4.  **验证**：检查在不同分辨率下的适配情况。
