## 1. 项目概述
BlackAd 是一个为媒体主（网站主）提供广告位管理、收益统计及数据分析的综合性平台。通过该平台，用户可以便捷地接入广告代码，实时监控广告表现，并进行多维度的收益分析。

## 2. 目标用户
- **媒体主 (User)**：管理自有网站、创建广告位、查看广告收益明细。
- **系统管理员 (Admin)**：审核媒体资质、管理系统用户、全局数据监控。

## 3. 功能模块详情

### 3.1 数字化仪表盘 (Dashboard)
- **核心价值**：提供业务全景图，快速了解收益动态。
- **主要功能**：
    - 关键指标展示：昨日收入、近7天收入、本月累计、日均预估。
    - 趋势图表：支持按维度（展现、点击、收入、点击率）切换查看波动趋势。
    - 活跃状态监控：显示当前待审核应用及系统运行状态。

### 3.2 媒体管理 (Media Management)
- **核心价值**：建立媒体身份标识，完成合规性审核。
- **主要功能**：
    - 媒体登记：支持域名、名称、分类、备案号录入。
    - 资质上传：支持代理授权书、软著扫描件上传。
    - 统计授权：支持百度统计/友盟统计的在线授权。
    - 状态流转：支持从“待初审”到“审核通过”的全生命周期管理。
- **[新增] 导入导出**：
    - **批量导出**：支持将媒体名称、域名、状态、创建时间等信息导出为 CSV/Excel。
    - **批量导入**：支持通过模板批量录入媒体信息，提高大客户接入效率。

### 3.3 代码位管理 (CodeSlots Management)
- **核心价值**：精细化控制广告展示形式，获取集成代码。
- **主要功能**：
    - 形式配置：支持 H5/PC 终端，固定块/悬浮/插屏等多种展示形式。
    - 尺寸自定义：支持 20:4 至 20:20 的比例范围设定。
    - 安全增强：提供反屏蔽代码选项（针对 UC/QQ 等第三方浏览器）。
    - 代码集成：一键生成 JS/HTML 集成代码，支持实时复制。
- **[新增] 导入导出**：
    - **批量导出**：支持按媒体或名称筛选后导出代码位配置信息。
    - **批量导入**：支持批量创建代码位，自动关联所属媒体。

### 3.4 数据统计与报表 (Data & Stats)
- **核心价值**：通过数据驱动决策，优化广告位表现。
- **主要功能**：
    - 数据概览：按日期范围筛选，展示展现量、点击量、CTR、eCPM、ACP 等核心指标。
    - 代码位数据：下钻至具体代码位的收益表现，支持调整系数（分成比例）查看。
    - 报表下载：支持将统计结果导出为 CSV 报表。

### 3.5 用户中心 (Users & Account)
- **核心价值**：确保系统访问安全，分权管理。
- **主要功能**：
    - 用户管理 (Admin)：创建账号、分配角色（管理员/普通用户）、重置密码。
    - 账号安全：用户可自主修改登录密码。

## 4. 技术架构
- **后端**：Java 17 + Spring Boot 3.4 + MyBatis-Plus + Spring Security (JWT)。
- **前端**：React 19 + TypeScript + Tailwind CSS 4 + Recharts。
- **存储**：MySQL 8.0 (核心业务) + Redis (缓存/鉴权)。
